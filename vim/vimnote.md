how to get vim help
-------------------------
1. :help {subject}    to show help doc
2. :helptags {dir}    generate help tags for directory {dir}

vim option
-------------------------
### Option Type
These options come in three forms:  
    --------------------------------------
    boolean   | can only be on or off
    number    | has a numeric value
    string    | has a string value
    --------------------------------------
